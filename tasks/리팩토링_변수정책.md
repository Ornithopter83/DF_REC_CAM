# 작업명

리팩토링_변수정책

## 목표

* var 변수들 중, 기본형에 근접한 변수를 명시적으로 선언하여 가독성을 높인다.
* 기본 자료형에 가깝고, 우항의 반환형이 절대 변하지 않는 float, int, boolean, string 자료형 등이 해당된다.

## 결과
* 기본형에 가까운 지역 변수 중 우항 타입이 명확한 `int`, `long`, `ulong`, `double`, `float`, `bool`, `string`, `TimeSpan`, `DateTime` 계열 선언을 명시 타입으로 변경했다.
* 화면 좌표, 크기, 비율, 파일 경로, 설정 문자열, 녹화 시간/용량 계산 등 사람이 직접 확인할 때 타입을 바로 알아야 하는 값 위주로 정리했다.
* 익명 타입, LINQ/XML 결과, WinForms 컨트롤 생성, OpenCV/이미지 리소스의 `using var`, 도메인 모델 결과처럼 `var`가 더 안전하거나 코드가 덜 어색한 경우는 유지했다.
* `dotnet build DFBlackbox\DFBlackbox.csproj -o .buildcheck` 결과 경고 0개, 오류 0개로 확인했다.
