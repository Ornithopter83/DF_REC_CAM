# 작업명

녹화/재생 전용 배포

## 목표

* 순수하게 녹화와 재생 관련된 기능만 보이고 사용할 수 있도록 숨기는 배포버전을 만든다.
* 숨겨지는 요소 : "차이 기준 저장" 버튼, "감시 시작 " 버튼,  "자동"라디오버튼,  설정창의 "감지"관련항목, "오버레이"관련항목 전부

## 관련 파일

* 이 작업으로 생성되는 파일은 없다.

## 변경 금지

* 기본 규칙을 따른다

## 요구사항

* 해당 배포버전의 트리거는 실행파일 명령 인수 --reconly 이다.
* UI 배치는 변경되지 않는다. 단지 비활성화가 아니라 아예 숨겨져야 한다.



## 완료 조건

* 컴파일 에러 없이 동작할 것

## 테스트 방법

* 

## 결과

* `--reconly`, `/reconly`, `-reconly` 실행 인수로 녹화/재생 전용 모드를 켜도록 추가했다.
* 녹화/재생 전용 모드에서 메인 화면의 `차이 기준 저장`, `감시 시작`, `자동` 라디오버튼을 숨겼다.
* 녹화/재생 전용 모드에서 설정창의 `감지`, `오버레이` 섹션을 숨겼다.
* 저장된 녹화 모드가 Auto여도 녹화/재생 전용 모드에서는 UI와 저장 로직이 Manual/Full만 사용하도록 보정했다.
