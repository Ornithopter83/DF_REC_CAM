# 결정 기록

이 문서는 프로젝트에서 결정된 사항만 짧게 기록한다. 대화 요약을 장문으로 남기지 않는다.

## 형식

```text
YYYY-MM-DD
- 결정 내용
- 이유 또는 영향
```

## 2026-06-26

- 긴 대화 이주용 요약을 반복 작성하지 않는다.
- 프로젝트 맥락은 대화가 아니라 저장소 문서에 둔다.
- 작업은 `tasks/*.md` 단위로 나누어 수행한다.
- 작업 완료 후 새로 생긴 결정사항만 `docs/DECISIONS.md`에 짧게 추가한다.
- 녹화/재생 전용 배포 모드는 실행 인수 `--reconly`로 켠다.
- 기본형에 가까운 지역 변수는 우항 타입이 명확하면 `var`보다 명시 타입을 우선 사용한다.

## 2026-07-15

- UI 외관 개선에는 Krypton Toolkit을 도입한다.
- 기능/성능 개선이 아니라 기본 Windows Forms 앱 느낌을 줄이는 차분한 밝은 테마 적용을 목표로 한다.
- 영상 표시, 녹화, 감지, 재생 흐름은 UI 테마 변경과 분리해 유지한다.
